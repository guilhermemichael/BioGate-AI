# BioGate AI e LGPD

## Natureza dos Dados

Dados biométricos são dados pessoais sensíveis e exigem tratamento reforçado.

## Requisitos Mínimos

- consentimento explícito
- finalidade específica
- base legal documentada
- registro de aceite
- possibilidade de revogação
- minimização de coleta
- retenção limitada
- trilha de auditoria
- controles de acesso

## Requisitos do Produto

- informar o motivo da captura
- informar o que será armazenado
- informar o prazo de retenção
- permitir exclusão quando aplicável
- registrar acessos administrativos

## Recomendações

- versionar o texto de consentimento
- manter política de retenção por tipo de dado
- separar dados identificáveis dos logs de uso
- revisar regularmente os acessos a dados sensíveis
