# Privacy Policy (LGPD)

BioGate AI respects user privacy and complies with data protection principles.

## Data Collected

- Name and email
- Authentication data
- Device metadata
- Biometric-derived data (processed representations, not raw media when possible)

## Purpose

Data is used for:
- Authentication
- Security analysis
- Risk scoring
- Audit logging

## Storage

Data is stored securely and protected against unauthorized access.

## User Rights

Users may:
- Request access to their data
- Request deletion
- Revoke consent

## Retention

Data is retained only as long as necessary for security and operational purposes.

## Security

We implement:
- Encryption
- Access control
- Logging and monitoring

## Contact

For data-related requests, contact the project maintainer.
